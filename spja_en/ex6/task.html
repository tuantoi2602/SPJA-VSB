SPJA Atoms Ex7

  • Your task is to create methods load_from_xml and __str__ in the class ExampleWorld.

    (4 points)

  • In the method load_from_xml(), parse the XML file atoms.xml. Set the size of drawing area (tag world in the XML file) and create the atoms according to the values in the XML file.
  • Save the atoms to the list.
  • HINT: for parsing the XML file, use the methods find('elem') or findall('elem')
    use .attrib['x'] for getting the element attribute, use .text for getting the value of element

    (1 point)

  • In the method main, add the command print(world). In the method __str__ of the class ExampleWorld, return a string describing the world and the atoms in the format:
     World size=500x500
   Number of normal atoms = 2
   Number of falldown atoms = 2
